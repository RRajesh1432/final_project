How It Works
4-STEP DETECTION PIPELINE
From raw audio file to threat verdict — runs in under 10 seconds on CPU
1
INPUT
🎵
Audio Input
Upload any file (WAV, MP3, FLAC, OGG, M4A, WebM) or record live from microphone. Supports continuous 10-second chunk monitoring for real-time surveillance.
2
PROCESS
📊
Spectrogram Conversion
Audio is resampled to 16 kHz mono, then converted to a Log-Mel Spectrogram using a 25ms Hanning window, 10ms hop, and 128 mel frequency bands — turning sound into an image.
3
CLASSIFY
🤖
AST Neural Network
The Audio Spectrogram Transformer (12 layers — first 6 frozen, last 6 fine-tuned on threat data) processes the spectrogram and outputs sigmoid probabilities for all 527 AudioSet classes.
4
DECIDE
⚡
Threat Decision + Alert
Score_t(x) = max probability over 5 threat classes. If Score_t ≥ τ (0.75), or a threat class appears in top-5 predictions — THREAT is flagged, ntfy fires instantly, all devices are notified.

THREAT CLASS SET — C_t
The 5 acoustic categories that trigger a threat alert
🔫
Gunshot
Firearm discharge, rifle, machine gun, fusillade
💥
Explosion
Blast, dynamite, grenade, boom, burst
🪟
Glass Break
Shatter, smash, crash, breaking window
😱
Scream
Shriek, shout, yell, wail, distress
🚨
Siren
Police, ambulance, fire truck, alarm
Each threat class maps to multiple AudioSet labels — e.g. "Gunshot" covers: gunshot, gunfire, machine gun, fusillade, artillery fire, cap gun, firearm, rifle, pistol.
READING THE THREAT SCORE
Score_t(x) = maximum sigmoid probability across C_t
0–29%
SAFE ✓
No threat class detected. Audio is speech, music, ambient noise, or other non-dangerous sound.
30–74%
MEDIUM ⚡
Possible threat pattern. Muffled or distant threat sound, or ambiguous recording. Review manually.
75–100%
HIGH ⚠
Score_t ≥ τ (0.75). Strong threat match. ntfy alert fires instantly. Treat as real until confirmed safe.
ⓘ 3 detection rules run in parallel: (1) Score_t ≥ τ=0.75, (2) threat class in top-5 predictions with score >0.15, (3) threat class is #1 or #2 prediction with score >0.10. Any rule firing triggers an alert.
UNDERSTANDING YOUR RESULTS
What each section on the Analyze page tells you
Threat Score Gauge
The circular dial shows Score_t — max probability across all 5 threat classes. The τ marker at 75% is the paper-validated detection threshold.
AI Explanation
Claude Haiku analyses the result and explains in plain English what was heard, how dangerous it is, and what action to take right now.
Log-Mel Spectrogram
Visual picture of the audio. Gunshots show sharp vertical lines; sirens show sweeping curves; explosions show wide energy bursts across all frequencies.
Temporal Segments
Audio split into 5 time windows — each classified independently. Shows exactly when in the recording the threatening sound occurred.
Threat Class Bars
Per-class probability for all 5 threat types. The tallest bar shows which category drove the alert — e.g. "gunshot: 82%".
Top AudioSet Classes
Top 12 predictions across all 527 AudioSet sound categories. Useful for understanding what non-threat sounds were also present in the recording.
